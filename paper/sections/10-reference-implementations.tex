\section{Current Implementation Coverage and Release Gaps}
\label{sec:reference-implementations}

This section records repository-observable coverage at source-tree revision
\texttt{83bf08d}, whose implementation code is unchanged here. It is not a limitation
on the survey taxonomy or a claim that the planned reference release exists. Statuses
mean current-tested, current-experimental, partial, or planned. Tests exist in the
repository, but this paper audit did not rerun them; the document build is not an
implementation test.

The present implementation is therefore useful evidence about a narrow planar
generator surface, not a reference benchmark. In particular, internal repair and
device-deterministic fixed-batch generation do not satisfy the raw-artifact lineage or
cross-platform reconstruction requirements. The planned release closes those gaps only
through the normative contracts in \cref{sec:benchmark-protocol}; its exact generator
set remains contingent on the screened taxonomy and engineering validation.
